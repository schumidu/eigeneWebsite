Grundlage

Mit besuch der Website haben Sie eingewilligt, dass die nötigsten, unten aufgeführten Daten von Ihnen gespeichert und verarbeitet werden. Dies ist technisch notwendig. Möchten Sie die Website besuchen ohne das auf Sie zurückführbare Informationen anfallen, nutzen Sie Anonymisierungsdienste wie z. B. Tor


Zweck

Um Ihren Besuch der Website möglichst benutzerfreundlich anbieten zu können, erheben wir eine Reihe von Daten und Informationen des Gerätes mit dem Sie diese Website aufgerufen haben. Dabei handelt es sich um folgende Daten:

  • IP-Adresse
  • Betriebssystem
  • Browsertyp und -version
  • Datum und Uhrzeit des Zugriffs
Eine Auswertung dieser Daten zu Marketingzwecken findet in diesem Zusammenhang nicht statt.


Dauer

Oben genannte Daten werden nur so lange wie nötig gespeichert. Maximal jedoch 6 Monate.

  • Recht auf Einschränkung der Verarbeitung
  • Wiederspruchsrecht
  • Recht auf Beschwerde bei einer Aufsichtsbehörde
  • Recht auf Datenübertragbarkeit
